# Mining Supply Chain Forecasting

Este repositorio contiene modelos de forecasting y análisis de inventario aplicados a supply chain minero. Incluye notebooks basados en datasets reales y técnicas utilizadas en la industria minera chilena.

## Contenido

- **M5 Forecasting – Starter Data Exploration**
  Exploración inicial de series temporales y técnicas de forecasting.

- **Demand Forecasting and Inventory Optimisation**
  Modelo completo de demanda y optimización de inventario.

## Tecnologías utilizadas
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- ARIMA, Prophet
- Jupyter Notebook / Kaggle
